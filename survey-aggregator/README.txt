Документация по работе API

1. Данные можно получить по ссылке http://survey-aggregator/api/survey/{id}
- данная ссылка обьявляется и обрабатывается в методе index() контролера Controller.php
- вместо параметра {id} можно вставить числовое значение любого индентификатора опроса
- данные приходят в формате json
- если опрос не опубликован или не существует, то в массиве json будет соответствующее сообщение

2. Все идентификаторы существующих опросов данного сервиса автоматически заносятся в массив $params в файле index.php
- данный массив вносится как параметр в метод getSurveyData() контролера APIController.php
- внутри этого метода также есть методы getSurveyData(), getAnswersData() которые прописаны в моделе Model.php

3. Также в методе getSurveyData() контролера APIController.php при return данных установлен флаг JSON_UNESCAPED_UNICODE.
Это сделано для того, чтобы избежать закодированных данных в получаемом json массиве
(например, если текст вопрос и ответы не на английском языке).
